
# Password Generator (Frontend Project)

A simple web-based Password Generator built using **HTML, CSS, and JavaScript**.  
This project allows users to either auto-generate a secure password or check the strength of a custom password.

---

## Features

### Mode Selection
Users can choose between:
- Auto Generate Password
- Create Custom Password

---

### Auto Password Generator
- Minimum password length: 12 characters
- User can select password length
- Random password generation
- Copy password button inside textbox
- Back navigation option

---

### Custom Password
- Enter your own password
- Password strength indicator
- Strength shown as:
  - Weak
  - Medium
  - Strong
- Percentage score displayed

---

### UI Features
- Background image support
- Centered container layout
- Styled buttons and inputs
- Responsive alignment
- Copy-to-clipboard functionality
